# 仕様書：クロコで初めてのWEBアプリワークショップ問診

## アプリ概要

| 項目 | 内容 |
|---|---|
| アプリ名 | クロコで初めてのWEBアプリワークショップ問診 |
| 用途 | ワークショップ参加者の事前環境確認 |
| 形式 | 単一HTMLファイル（外部依存なし） |
| 対象デバイス | PC・スマートフォン対応 |

## 設問フロー

```
[前提確認] クロコのチャットができているか？
    │
    ├─ できていない・不安 → ブロックメッセージ表示（終了）
    │
    └─ できている
           │
           └─ OS・CHK1〜5 を一気に全表示
                  ├─ OS選択（Mac / Windows）
                  ├─ CHK1：フォルダ作成
                  ├─ CHK2：スクリーンショット
                  ├─ CHK3：GitHubアカウント
                  ├─ CHK4：Cloudflareアカウント
                  └─ CHK5：学長プロンプト実行
                               │
                               └─ 全問回答 → 結果コピー
```

## 各設問の詳細

### 前提確認
- **質問**：クロコ（Claude Codeデスクトップ版）を起動して、チャットが普通にできていますか？
- **選択肢**：できている（緑）／ できていない・不安（オレンジ）
- **NG時**：赤アラートでブロック。以降の設問・OS選択は非表示のまま

### OS
- **質問**：パソコンの種類を教えてください。
- **選択肢**：Mac（緑）／ Windows（緑）
- **目的**：コピー結果への記録のみ（CHK2ヒントはMac/Windows両方を常時表示）

### CHK 1：フォルダ作成
- **質問**：クロコのcodeで「デスクトップに"オフ会作業フォルダ"というフォルダを作って下さい」と送って、フォルダが作れましたか？
- **選択肢**：できた！（緑）／ できなかった（オレンジ）
- **NG時**：手順アドバイスを表示。ブロックはしない

### CHK 2：スクリーンショット
- **質問**：スクリーンショット（画面の一部撮影）の撮り方をマスターしていますか？
- **ヒント常時表示**：Mac: Command＋Shift＋4 ／ Windows: Windows＋Shift＋S
- **選択肢**：マスターした！（緑）／ まだ不安（オレンジ）
- **NG時**：両OS対応のヒントを表示。ブロックはしない

### CHK 3：GitHubアカウント
- **質問**：GitHub のアカウントを持っていますか？
- **選択肢**：ある（緑）／ ない（オレンジ）
- **NG時**：「オフ会前に作成を。当日一緒にやることもできます」と表示

### CHK 4：Cloudflareアカウント
- **質問**：Cloudflare のアカウントを持っていますか？
- **選択肢**：ある（緑）／ ない（オレンジ）
- **NG時**：「オフ会前に作成を。当日一緒にやることもできます」と表示

### CHK 5：学長プロンプト
- **質問**：「学長プロンプト」を実行しましたか？
- **補足**：リベシティで両学長が配布しているプロンプト。クロコのチャットに貼り付けてClaude.mdにセキュリティ設定を追記する
- **選択肢**：やった！（緑）／ まだ（オレンジ）
- **NG時**：手順案内を表示。ブロックはしない（当日対応可）

## 結果コピー機能

- 全問（7問）回答するとコピーボタンが有効化
- コピー内容の形式：
```
【クロコWEBアプリワークショップ 事前問診】

前提. クロコ起動:             できている
OS:                           Mac
CHK1. フォルダ作成:           できた
CHK2. スクリーンショット:     マスターした
CHK3. GitHub:                 ある
CHK4. Cloudflare:             ある
CHK5. 学長プロンプト:         やった
```
