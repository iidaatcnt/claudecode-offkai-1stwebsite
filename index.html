Q3
Mac の macOS バージョンを教えてください。
（左上リンゴ → このMacについて で確認できます）
Claude Code の動作には macOS 13 Ventura 以降 が必要です。
できればオフ会前にアップデートをお試しください（時間がかかる場合があります）。
お使いの Mac では Claude Code が動作しない可能性があります。
事前にサバ缶へご相談ください。
Q3
Windows のバージョンを教えてください。
（スタートボタン右クリック → システム で確認できます）
Windows 10 でも動作しますが、Windows 11 へのアップグレードを推奨します。
2025年10月にサポートが終了予定です。
Windows 8 以前では Claude Code は動作しません。
事前にサバ缶へご相談ください。
Q4

ChatGPT の無料アカウントが必要です。
オフ会前に chatgpt.com でアカウントを作成しておいてください（メールアドレスで登録できます）。
Q5
Cloudflare のアカウントを持っていますか？

オフ会前に cloudflare.com でアカウントを作成しておいてください。
メールアドレスだけで登録でき、クレジットカードは不要です。
