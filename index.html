前提確認
クロコ（Claude Codeデスクトップ版）を起動して、チャットが普通にできていますか？
このオフ会は クロコ（Claude Code）のインストール済みの方 が対象です。
まずはクロコインストール伴走会へご参加ください。
インストールが完了してからまたご確認ください。
CHK 1
クロコのcodeで「デスクトップに"オフ会作業フォルダ"というフォルダを作って下さい」と送って、フォルダが作れましたか？
※まだの場合は今すぐ試してみてください！
クロコのcode欄に「デスクトップに"オフ会作業フォルダ"というフォルダを作って下さい」と入力してみてください。
うまくいかない場合はサバ缶へご連絡ください。当日一緒に確認しましょう。
CHK 2
スクリーンショット（画面の一部撮影）の撮り方をマスターしていますか？
Mac：Command ＋ Shift ＋ 4　／　Windows：Windows キー ＋ Shift ＋ S
Mac は Command ＋ Shift ＋ 4、Windows は Windows キー ＋ Shift ＋ S で画面の一部を撮影できます。
オフ会前に一度試しておいてください。
CHK 3
GitHub のアカウントを持っていますか？
当日、Webサイトのファイルを管理するのに使います（無料）
オフ会前に github.com で無料アカウントを作成しておいてください。
メールアドレスだけで登録できます。
CHK 4
Cloudflare のアカウントを持っていますか？
当日、Webサイトをインターネットに公開するのに使います（無料・カード不要）
オフ会前に cloudflare.com で無料アカウントを作成しておいてください。
メールアドレスだけで登録でき、クレジットカードは不要です。
CHK 5
「学長プロンプト」を実行しましたか？
リベシティで両学長が配布しているプロンプトです。クロコのチャットに貼り付けて送信すると、Claude.md にセキュリティ設定が追記されます。
リベシティから配布されている学長プロンプトをクロコのチャット欄に貼り付けて送信してください。
Claude.md にセキュリティ設定が自動で追記されます。
オフ会当日に一緒にやることもできますので、わからなければそのまま来てください！
