# クロコで初めてのWEBアプリワークショップ問診

Claude Code（クロコ）デスクトップ版インストール済みの方を対象に、
「クロコで初めてのWEBアプリワークショップ」の事前確認を行う問診Webアプリです。

## 概要

- **対象**：Claude Code（クロコ）インストール済みで、初めてWebサイトを公開したい方
- **目的**：ワークショップ当日にスムーズにWebサイト公開できるよう、事前に環境・スキルを確認する
- **ブロック機能**：クロコが使えていない人はCHK設問を見せない

## 当日のゴール

1. クロコの基本操作（フォルダ作成・ファイル操作）を習得
2. Claude Code でHTML/CSSをコーディング
3. GitHub にリポジトリを作成してコードを管理
4. Cloudflare Pages にデプロイ → スマホでも見られるURLで公開

## 設問構成

| No | 内容 | 「NG」時の動作 |
|---|---|---|
| 前提確認 | クロコでチャットができているか | **ブロック**（以降の設問を非表示） |
| OS | Mac / Windows の選択 | — |
| CHK 1 | クロコでデスクトップにフォルダが作れたか | アドバイス表示 |
| CHK 2 | スクリーンショットをマスターしているか | ヒント表示（Mac/Windows両方） |
| CHK 3 | GitHub アカウントの有無 | 作成案内表示 |
| CHK 4 | Cloudflare アカウントの有無 | 作成案内表示 |
| CHK 5 | 学長プロンプトを実行したか | 手順案内表示 |

## UX仕様

- 前提確認「できている」→ OS選択 + CHK1〜5 を一気に全表示
- 前提確認「できていない・不安」→ 以降をブロック
- 選択後でも押し直して変更可能
- 「最初からやり直す」ボタンあり（ページリロード）
- 全問回答でコピーボタンが有効になり、結果をテキストでコピーできる

## ボタン色のルール

- **緑**：準備OK・推奨の選択肢
- **オレンジ**：要確認・事前準備が必要な選択肢

## デザイン

- ベースカラー：クロコオレンジ（`#DA7756`）
- スマートフォン対応（480px以下でボタン縦並び）
- 単一ファイル構成（`index.html`）・外部依存なし

## デプロイ先

- **URL**：https://claudecode-offkai-1stwebsite.pages.dev/
- **方法**：GitHub への push で Cloudflare Pages が自動デプロイ
